# Type Ia Supernova Observables

This directory tracks source manifests and validation-set definitions
for Type Ia supernova observational data.

The first target is not "all data copied into one place." The first
target is a reviewable evidence graph:

- object identity from TNS and source aliases,
- photometry and spectra from authoritative archives and survey
  releases,
- publication and data-release provenance,
- calibration and filter-system metadata,
- selection cuts and quality cuts,
- simulation-facing validation products.

## Initial catalog classes

- **Identity and classification:** TNS.
- **Spectra and related files:** WISeREP.
- **Cosmology compilations:** Pantheon+ / SH0ES.
- **Survey light curves:** DES-SN 5YR, ZTF Type Ia DR1, SDSS-II SN.
- **Low-redshift calibration:** CSP DR3, Foundation, CfA, CNIa0.02.

## Validation families

- Cosmology distance products and covariance.
- Nearby calibration-quality multi-band light curves.
- Spectral velocity and feature-evolution benchmarks.
- Rate, host, and population-distribution summaries once source
  selection functions are represented.
